# Tic-Tac-Toe-by-Liza

### **_Introduction_**
This application is a tic-tac-toe game for 2 players. The first player to 3 marks in a straight line wins the game.

### **_Used Technologies_**
Python
Jupiter Notebook
Anaconda
 
## How to run the game
Run the "Tic Tac Toe by Liza.ipynb" file in a Jupiter Notebook.
